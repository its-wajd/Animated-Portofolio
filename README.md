# Wajd Dev Portfolio

A modern, animated, and performance-focused developer portfolio built with pure **HTML, CSS, and JavaScript**.

This portfolio presents my work, skills, projects, and contact information in a clean premium dark interface with smooth scrolling animations, interactive UI elements, and a security-focused developer identity.

## 🧩 Customization

You can edit the main content directly from:

```txt
index.html
```

You can edit the design and animations from:

```txt
styles.css
```

You can edit interactions and scrolling behavior from:

```txt
script.js
```

## 🌐 Deployment

You can deploy this portfolio easily on:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages
- Any shared hosting service

For GitHub Pages:

1. Push the project to GitHub.
2. Go to repository settings.
3. Open **Pages** settings.
4. Select the main branch.
5. Save and open your generated link.
